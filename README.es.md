## 🚀 Inicio rápido

### Requisitos previos

- PHP 8.2+
- Composer
- MySQL 8+

### Instalación

```bash
git clone https://github.com/flacks-cc/api-laravel.git
cd api-laravel
composer install
php artisan serve
```
